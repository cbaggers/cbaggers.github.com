    • ( SWIZZLE VEC PATTERN )

      accessor

      Swizzle allows you to compose a new vector out of an existing one by picking it's components.

      Given a vec3 called a = (v! 1 2 3)

      • (swizzle a :xy) â (v! 1 2)
      • (swizzle a :xx) â (v! 1 1)
      • (swizzle a :yx) â (v! 2 1)
      • (swizzle a :yyyy) â (v! 2 2 2 2)
      • (swizzle a :zzyy) â (v! 3 3 2 2)

    • ( S~ VEC PATTERN )
